Next.js App Router 예약 파일명 (모두 소문자)
페이지 관련

page.tsx - 페이지 컴포넌트
layout.tsx - 레이아웃 컴포넌트
template.tsx - 템플릿 컴포넌트

에러 처리

error.tsx - 에러 바운더리
global-error.tsx - 글로벌 에러
not-found.tsx - 404 페이지

로딩 및 기타

loading.tsx - 로딩 UI
route.tsx - API 라우트
middleware.tsx - 미들웨어

메타데이터

sitemap.xml - 사이트맵
robots.txt - 로봇 파일
